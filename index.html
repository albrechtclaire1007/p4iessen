OUTPUT: Return ONE full HTML file only. No explanations, no extra text. 🧠 Game prompt suggestions (choose one per chapter/exercise) These are “mini-prompts” they can paste into ChatGPT after the layout prompt, to generate the exercise logic without changing the layout. 1) Lauschter a klick (audio → pick correct) Create a “listen and click” exercise. DATA: items = [{word, img, audio}] Flow: play target audio → learner clicks the correct image → green flash + next item; wrong = red flash. Shuffle items each reset. Connect play button to replay current audio. Connect replay button to reshuffle and restart. 2) Memory (image ↔ word OR image ↔ audio) Create a memory game with 8 pairs (or fewer if less data). Pairs: either image+word OR image+audio. Rules: flip 2 cards only; lock while checking; match stays open; mismatch flips back. Replay button = reshuffle/reset. Optional: play audio when revealing the word/audio card. 3) Drag words into sentence (word tiles → slots) Create a drag/click “build the sentence” game. DATA: sentences = [{tokensInOrder:[...], distractors:[...optional]}] Show empty slots, provide shuffled tokens below. Learner drags or taps tokens to fill slots. Allow removing a placed token by tapping the slot. Check correctness when all slots filled; if correct → next sentence. Replay = restart current sentence with reshuffle. 4) Multiple choice quiz (1 correct + 3 distractors) Create a multiple choice exercise. DATA: questions = [{prompt, options:[...], correctIndex, audio(optional), img(optional)}] Show prompt (and image if present), 4 option buttons. Correct = green flash + next question; wrong = red flash, allow retry. Play = play question audio if available. Replay = restart current question. 5) Sort into categories (drag/tap into columns) Create a sorting game (drag & drop + tap fallback). DATA: items=[{img/word, cat}], categories=[{id,label}] Show 2–4 category columns and a pool of items. Correct drop moves into column; wrong flashes red and returns. Replay = reset and reshuffle. 6) Conjugation picker (infinitive → choose correct form) Create a conjugation exercise. DATA: prompts=[{pronoun, infinitive, options:[...], correct, sentenceTemplate(optional)}] Show pronoun + infinitive and 3–5 options as pills/buttons. Learner selects; correct = next; wrong = retry. Optional: after correct, show full sentence briefly. Replay = reshuffle and restart. 7) Inversion / question order (P4/P5 style) Create an inversion game (build a question). DATA: prompts=[{statementTokens:[...], questionTokensCorrect:[...]}] Show scrambled question tokens; learner orders them into slots. Check exact order; correct = next. Replay = reshuffle current. 8) Gap-fill (fill the missing word) Create a gap-fill exercise. DATA: prompts=[{sentenceParts:["...", "___", "..."], options:[...], correct}] Display sentence with a blank and options. Click option fills blank; correct = next; wrong = retry. Replay = reset.
